A Google Keep-like note-taking application for WordPress.

== Description ==

RatNotes is a simple, intuitive note-taking plugin that brings the familiar Google Keep experience to your WordPress dashboard.

**Features:**

* Create and edit notes quickly
* Color-code notes with 9 different colors
* Pin important notes to the top
* Archive notes you want to keep but don't need immediately
* Trash functionality with restore option
* Search notes by title or content
* Responsive design for mobile and desktop
* Keyboard shortcuts for power users

== Installation ==

1. Upload the `ratnotes` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'RatNotes' in the admin menu
4. Start creating notes!

== Frequently Asked Questions ==

= Where are my notes stored? =

Notes are stored in your WordPress database in a dedicated table. Each user's notes are private to that user.

= Can I export my notes? =

Currently, export functionality is not available. This is planned for a future release.

= Are notes synced across devices? =

Yes! Since notes are stored in your WordPress database, you can access them from any device by logging into your WordPress admin.
